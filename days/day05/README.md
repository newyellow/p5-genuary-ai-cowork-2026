# Day 05 - Write "Genuary" (Avoid using a font)

**Prompt:** Write "Genuary". Avoid using a font.

**Concept:** Collective Message — 個體組成字母，共同拼出意義

## 核心概念

### 人形組成文字

不使用字體，而是用數百個微小的人形排列組成 "GENUARY" 七個字母。每個字母由 7-16 個人組成，總共約 80 個人形。

### 從個體到集體意義

**湧現的訊息**
- 單一個體：只是一個小人
- 多個個體：形成字母的輪廓
- 所有個體：拼出完整的訊息 "GENUARY"

這完美展現了**湧現（Emergence）**的概念：
- 個體很小，看似微不足道
- 但當他們聚集、組織、合作
- 突然間，意義出現了

### Meta 敘事：共同的目標

**我們為什麼在這裡？**
- 80 個人形，每個都在自己的位置
- 他們可能不知道自己在拼什麼字
- 但從更高的視角看，他們共同拼出 "GENUARY"
- 這就是我們的目標，我們的共同目的

**個人努力 + 集體意義**
- 每個人在自己的位置上微微移動、呼吸、伸展
- 個體的小動作不會破壞整體的字形
- 反而讓文字「活」了起來
- 靜態的文字變成動態的生命體

## 技術實作

### 字母點陣定義

使用簡化的幾何形式定義每個字母：
```javascript
G: [[1,0],[2,0],[3,0],...] // 13 points
E: [[0,0],[0,1],...] // 15 points
N, U, A, R, Y: ... // 各 7-16 points
```

### 視覺效果

**人形配置**
- 每個字母用不同色相（hue 200-320）
- 同一字母內的人形色相相近但有變化
- 大小略有差異（scale 0.4-0.6）
- 頭部形狀隨機（圓、方、三角）

**動畫**
- **微小軌道運動**：每個人繞著基礎位置做圓形軌道（半徑 2-5px）
- **手臂擺動**：向上伸展，象徵向上努力
- **身體旋轉**：輕微的搖擺
- **連接線**：週期性顯示同字母內人形的連接（展現結構）

### 湧現效果

- 80 個獨立的人形
- 每個都有自己的運動相位和速度
- 個體的隨機性不破壞整體可讀性
- 反而創造出「活字」的效果

## 互動

- **按 'S' 鍵**：儲存當前畫面
- **連接線**：每 2 秒週期性顯示/隱藏

## 哲學反思

### 意義的建構

這件作品探討**意義是如何從個體建構出來的**：

1. **個體層次**：每個人只是一個小點
2. **局部層次**：幾個人形成一個字母的筆畫
3. **整體層次**：所有人共同拼出 "GENUARY"
4. **Meta 層次**：這個字代表我們共同的挑戰和目標

### 無字體的書寫

不使用字體意味著什麼？
- 不依賴預設的系統
- 自己定義每個字母的形狀
- 用生命（人形）而非死物（字型）來書寫
- 文字不再是靜態的，而是活著的

### Genuary 的意義

為什麼要拼出 "GENUARY"？
- 這是我們 31 天旅程的名字
- 每個人（創作者）都是這個字的一部分
- 我們共同組成這個挑戰的意義
- 個人的努力貢獻於集體的成就

> "Alone we are letters, together we spell hope."

## 技術細節

- 字母點陣系統
- 簡化的幾何形狀（4-5 個單位寬）
- 有機的位置擾動（±5px）避免過於機械
- 基於相位的獨立動畫
- 連接線演算法：距離 < 40px 且同字母
