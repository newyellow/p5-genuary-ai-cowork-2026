# Day 01 - One color, one shape

**Prompt:** One color, one shape

## 概念

使用最簡約的元素創作：
- **一種形狀**：圓形（Circle）
- **一種顏色**：藍色（Hue 210）

透過調整圓形的大小、亮度和透明度，創造出豐富的視覺層次。圓形會緩慢移動並生長/縮小，產生有機的動態感。

## 互動

- **點擊滑鼠**：在滑鼠位置新增圓形
- **按 'R' 鍵**：重置畫面
- **按 'S' 鍵**：儲存當前畫面為 PNG

## 技術細節

- 使用 p5.js 的 HSB 色彩模式
- 透過 alpha 值的漸層疊加產生視覺深度
- 每個圓形有自己的移動速度和生長速度
- 背景有淡淡的軌跡效果
