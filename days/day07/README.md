# Day 07 - Boolean Algebra

**Prompt:** Boolean algebra - Get inspired by Boolean algebra in any way

**Concept:** Binary Existence — 一切被簡化為真/假、1/0

## 核心概念

120 個人形排列成網格，每個人被兩個布林變數（A 和 B）控制。透過邏輯運算（AND, OR, XOR, NOT），決定他們的顏色、大小、姿態和位置。

### 布林運算的視覺化

- **A OR B** → 顏色（至少一個為真則明亮）
- **A AND B** → 大小（兩者都為真則變大）
- **A XOR B** → 手臂（互斥為真則舉手）
- **NOT A** → 位置（否定則上升）

### Meta 敘事：二元分類系統

**真/假的世界**
- 每個人的狀態由外部邏輯決定
- 不是 A 就是 B，沒有中間地帶
- 複雜的人被簡化為二進位數據

**系統性控制**
- 布林邏輯 = 社會分類系統
- 你屬於哪個類別？A 還是 B？
- 內/外、我們/他們、對/錯

## 哲學

> "In a binary world, nuance disappears. We are all reduced to TRUE or FALSE."
