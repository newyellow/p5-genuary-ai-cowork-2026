# Genuary 2026 Prompts

Genuary 是一個為期一個月的生成藝術挑戰，每天有一個創作題目。

官方網站: https://genuary.art/prompts

---

## 完整題目列表

### JAN.1 - One color, one shape
使用一種顏色、一種形狀進行創作

### JAN.2 - Twelve principles of animation
[十二個動畫原則](https://en.wikipedia.org/wiki/Twelve_basic_principles_of_animation) - 探索動畫的基本原理

### JAN.3 - Fibonacci forever
永恆的斐波那契 - 創作一個以某種方式使用[斐波那契數列](https://en.wikipedia.org/wiki/Fibonacci_sequence)的作品

### JAN.4 - Lowres
低解析度 - 創作一個細節被簡化或像素化的低解析度圖像或圖形

### JAN.5 - Write "Genuary". Avoid using a font
書寫 "Genuary" 字樣，但避免使用字體

### JAN.6 - Lights on/off
燈光開關 - 創作一個在開關「數位」燈光時會改變的作品

### JAN.7 - Boolean algebra
布林代數 - 以任何方式從[布林代數](https://en.wikipedia.org/wiki/Boolean_algebra)獲得靈感

### JAN.8 - A City
一座城市 - 創造一個生成式的大都會

### JAN.9 - Crazy automaton
瘋狂自動機 - 具有瘋狂規則的細胞自動機

### JAN.10 - Polar coordinates
極座標系統

### JAN.11 - Quine
[Quine](https://en.wikipedia.org/wiki/Quine_(computing)) - 一種程式碼詩歌形式，是一個輸出自己源代碼的程式

### JAN.12 - Boxes only
只用盒子/方塊

### JAN.13 - Self portrait
自畫像 - 創造一個代碼生成的自畫像

### JAN.14 - Everything fits perfectly
一切完美契合

### JAN.15 - Create an invisible object where only the shadows can be seen
創造一個看不見的物體，只能看到它的陰影

### JAN.16 - Order and disorder
秩序與混亂

### JAN.17 - Wallpaper group
[壁紙群組](https://en.wikipedia.org/wiki/Wallpaper_group) - 只有 17 種方式可以用重複圖案覆蓋平面，選擇你最喜歡的一種

### JAN.18 - Unexpected path
意外的路徑 - 繪製一條基於一個非常簡單規則而改變方向的路線

### JAN.19 - 16x16
16x16 網格或解析度

### JAN.20 - One line
一條線 - 創作一個只由單一線條組成的作品

### JAN.21 - Bauhaus Poster
包浩斯海報 - 創作一個受德國藝術學校[包浩斯](https://en.wikipedia.org/wiki/Bauhaus)啟發的海報設計

### JAN.22 - Pen plotter ready
適合筆繪機輸出的作品

### JAN.23 - Transparency
透明度 - 探索透明的概念

### JAN.24 - Perfectionist's nightmare
完美主義者的噩夢

### JAN.25 - Organic Geometry
有機幾何 - 看起來或行為像有機物，但完全由幾何形狀構成的形式

### JAN.26 - Recursive Grids
遞迴網格 - 將畫布分割成某種網格，並在每個單元格中一次又一次地遞迴

### JAN.27 - Lifeform
生命形式 - 一個行為像是活著或正在生長的形狀或結構

### JAN.28 - No libraries, no canvas, only HTML elements
不使用函式庫、不使用 canvas，只用 HTML 元素

### JAN.29 - Genetic evolution and mutation
遺傳演化與突變

### JAN.30 - It's not a bug, it's a feature
這不是 bug，這是特性

### JAN.31 - GLSL day
GLSL 日 - 只使用著色器創作作品

---

## 注意事項

- 題目不需要完全按照提示執行，可以自由詮釋
- 可以使用任何語言、框架或媒介
- 可以完成全部或部分題目
- 使用標籤 #genuary 和 #genuary2026，以及 #genuary1, #genuary2 等對應日期的標籤
